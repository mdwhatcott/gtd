## A Scoped Approach to Naming Conventions in this Project


## Within func declarations:

    var/const   NAME or nameNAMEname
    receiver    `this`
    input       _name or `input`
    output      name_ or `result(s)`


## Within package declarations:

    var/const   N/name
    type        N/name
    method      N/name
    field       N/name
